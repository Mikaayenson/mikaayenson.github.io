NIST Risk Analysis Framework on Web Servers

The National Institute of Standards and Technology has provided specific guidelines on securing web servers. NIST 800-44 was designed as a framework for, “federal departments and agencies, state agencies, and commercial organizations in installing, configuring, and maintaining secure public Web servers. It presents generic security principles and covers details specific to the various components of Web content, Web applications, and Web servers (Tracy, Jansen & LcLarnon, 2017).”

The framework recommended mitigation strategies on web server attacks consist of a large checklist provided by NIST 800-44 Appendix E.

NIST Actionable Web Server Attack Mitigation Guidelines

Tracy, M., Jansen, W., & LcLarnon, M. (2017). Guidelines on Securing Public Web Servers. Gaithersburg: National Institute of Standards and Technology.
